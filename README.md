# Sales_Performance_Target_Achievement_Dashboard
## Project Overview
This Project is an interactive Sales Performance Dashboard created in Microsoft Excel to analyze Sales executive peformance, target achievement, and sales trends.

The dashboard helps management track key sales metrics and identify high-performing and low performing executive for better decision-making.
